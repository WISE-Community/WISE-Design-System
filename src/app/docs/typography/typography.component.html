Typography

Overview

WISE uses the Tailwind CSS Typography plugin with customizations to define typography styles that are used throughout WISE applications. To utilize the typography styles demonstrated in this document, add the wisetype, dark:wisetype-dark, and wisetype-neutral classes to the <body> or a parent DOM element.

Typographic content can also be modified using any Tailwind CSS typography utility classes. Check out the Tailwind CSS documentation for more information.

Font Families

WISE uses two font families by default - one for body content and one for headings. They can also be applied to any HTML element using the following classes:
